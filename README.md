# Daily Perlman

A responsive comic gallery built with React + TypeScript, Vite, React-Router, and Tailwind CSS.

The app lets you browse comics, filter and search them, view full details, and save favorites locally.

This project was built strictly as a personal project with no intention of commercial use. Just a fan of Asher Perlman comics!

Everything is done client side here. Comics are loaded from local json. Favorited comics are thrown in localStorage.

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
- Tailwind CSS v4

### Prerequisites

- Node.js 20+ (or current LTS)
- npm

### Install

```bash
npm install
```

### Run in development

```bash
npm run dev
```

Then open the local URL shown in your terminal (usually `http://localhost:5173`).
